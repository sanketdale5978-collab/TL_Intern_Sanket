03 — Project

Project Showcase

A complete AI-powered medical device — from dataset to real-time deployment.

🔬 AI Healthcare · Raspberry Pi 4
Real-Time Skin Cancer Detection
Using Deep Learning on Raspberry Pi 4

An end-to-end AI system that captures live skin images via Raspberry Pi Camera, runs MobileNetV2 / EfficientNetB0 inference in real-time, and classifies lesions as Benign or Malignant — inside a custom 3D-printed enclosure designed in Fusion 360.

Software Architecture
  • ISIC 2018 dataset — 10,000+ dermoscopic images
  • RMSE-based class balancing for fairness
  • ImageDataGenerator with augmentation pipeline
  • Transfer Learning — MobileNetV2 + EfficientNetB0
  • Custom classification head with Dense + Softmax
  • TFLite model conversion for Pi deployment
  • OpenCV real-time camera inference loop
Hardware Architecture
  • Raspberry Pi 4 (4GB RAM) as inference engine
  • Raspberry Pi Camera Module v2 — 8MP
  • TensorFlow Lite runtime for Pi optimization
  • Custom 3D-printed enclosure (Bambu Lab H2D)
  • Autodesk Fusion 360 CAD design workflow
  • Real-time classification output on display
  • Low-latency edge inference pipeline
